# Suivi TSX

Page statique (aucun backend) pour suivre XEQT, ZEB, CHPS et VDY (TSX) :
cours, composition de chaque FNB, et un bilan technique automatique
(Achat en bloc / DCA / Attends / Vends).

## Utilisation

Ouvrir `index.html` (ou la page hébergée sur GitHub Pages), coller une clé
API gratuite [Twelve Data](https://twelvedata.com/) dans le champ prévu
(stockée uniquement dans le navigateur, jamais envoyée ailleurs), puis
cliquer « Actualiser ».

⚠️ Outil personnel d'aide à la décision — pas un conseil financier. Voir la
section « Méthodologie » en bas de la page pour le détail du calcul du
bilan. Les compositions des FNB sont des instantanés figés : vérifier la
fiche officielle de chaque émetteur pour les pondérations à jour.

Suivi TSX — XEQT · ZEB · CHPS · VDY

Cours, composition et bilan technique (achat en bloc / DCA / attends / vends) pour quatre FNB cotés au TSX.

⚠️ Outil personnel d'aide à la décision, pas un conseil financier. Le bilan est calculé automatiquement à partir de quatre indicateurs techniques simples (voir « Méthodologie » en bas de page) — il ne remplace pas une analyse complète ni les conseils d'un professionnel. Prix et historique viennent tous les deux de Yahoo Finance (endpoint non officiel, quasi temps réel, relayé via un proxy CORS tiers non affilié — allorigins.win — car Yahoo bloque les appels directs depuis un navigateur) : aucune clé API requise, mais aucune garantie officielle non plus — ça peut cesser de fonctionner sans préavis. Les compositions des FNB sont des instantanés figés — vérifiez toujours la fiche officielle de l'émetteur (lien dans chaque carte) pour les pondérations à jour.
Méthodologie du bilan (Achat en bloc / DCA / Attends / Vends)

Le bilan additionne les points de 4 indicateurs techniques calculés à partir de l'historique quotidien (~1 an) de chaque titre, puis classe le score total :

Score total (de −7 à +8) →

≥ 5🟢 Achat en blocRabais marqué + tendance de fond intacte : conviction élevée pour un dépôt important.

≤ −3🔴 VendsTendance de fond cassée et momentum négatif : signal tactique de réduction, pas une recommandation d'abandonner un placement de long terme.

Ce système est simpliste par construction (aucune analyse fondamentale, aucun contexte macro) — il sert de repère, pas de vérité absolue. Pour XEQT et VDY en particulier (FNB de long terme largement diversifiés), un signal « Vends » devrait être rare et interprété avec prudence.

Données de marché : Yahoo Finance (endpoint non officiel, relayé par le proxy tiers allorigins.win), aucune clé requise. Compositions des FNB : instantanés manuels, voir liens officiels dans chaque carte. Page statique sans backend — tout le calcul se fait dans le navigateur.
